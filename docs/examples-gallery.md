# Example PDFs

These artifacts are generated by running `python examples/generate_examples.py` (the docs CI runs this script before building the site). Use the links below to download the latest Markdown/PDF outputs for each runnable example.

| Example | Description | PDF | Markdown |
| --- | --- | --- | --- |
| Simple | Minimal report with one section and table (`examples/simple/builder.py`). | [PDF](examples/generated/simple.pdf) | [Markdown](examples/generated/simple.md) |
| Complex | Rich blocks (bullets, checklist, code, math, image) (`examples/complex/builder.py`). | [PDF](examples/generated/complex.pdf) | [Markdown](examples/generated/complex.md) |
| Simple Playbook | End-to-end quarterly snapshot with mixed layouts (`examples/playbook/simple_full.py`). | [PDF](examples/generated/simple_full.pdf) | [Markdown](examples/generated/simple_full.md) |
| IEEE Sample | IEEE-style paper rendered with `IEEETemplate` (`examples/pdf/ieee_builder.py`). | [PDF](examples/generated/ieee_sample.pdf) | [Markdown](examples/generated/ieee_sample.md) |
